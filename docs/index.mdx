# Introduction

Superface is an AI-powered developer toolkit made up of a CLI and robust SDK that make creating API integrations faster, and more manageable by helping you think about _the use case_ you want to achieve, _not_ how you need to write the code for the API to achieve it, allowing you to focus on developing what matters.

## Why Superface?

Superface can be mad beneficial! Here are four reasons why we think you should try it.

<div className="col col--6">

    <div className="card shadow">

        </p>
      </div>
    </div>
  </div>
    <div className="col col--6">
    <div className="card shadow">
      <div className="card__body">
        <h3>üßê Abstracted Integrations</h3>
