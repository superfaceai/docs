# Formal specifications

<div class="row padding-bottom--lg">
  <div class="col col--6">
    <div class="card shadow">
      <div class="card__body">
        <h3>Comlink Profile</h3>
        <p>

        </p>
      </div>
      <div class="card__footer">
        <a href="/docs/comlink/profile" class="button button--primary button--block">Profile (language spec)</a>

  <div class="col col--6">

      <div class="card__body">
        <h3>Comlink Map</h3>
        <p>
          Comlink Map is a format for describing one concrete implementation of a Comlink Profile. It essentially maps the application (business) semantics into provider’s interface implementation.
        </p>
      </div>
      <div class="card__footer">
        <a href="/docs/comlink/map" class="button button--primary button--block">Map (language spec)</a>
      </div>
    </div>
  </div>
</div>

<div class="row">

        </p>
      </div>
      <div class="card__footer">
        <a href="/docs/comlink/provider.yaml" class="button button--primary button--block">Provider JSON schema (YAML)</a>
